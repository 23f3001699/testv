## Devcontainer Setup

This repository includes a devcontainer configuration for Verilog development with the OSS CAD Suite. See [.devcontainer/README.md](.devcontainer/README.md) for details.

To get started:
1. Open this repository in VS Code
2. Install the "Dev Containers" extension
3. Click "Reopen in Container" when prompted
4. The OSS CAD Suite will be automatically installed and configured
